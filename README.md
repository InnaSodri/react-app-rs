# React App RS

This project uses Vite and React. To run locally, create a `.env` file in the project root and set `VITE_API_KEY` with your The Movie Database API key.

```bash
VITE_API_KEY= '4e44d9029b1270a757cddc766a1bcb63';
```

Then install dependencies and start the dev server:

```bash
npm install
npm run dev
```
